# Banking Complaint Analytics & Management System (SQL Project)

## Overview

This project implements a relational database system to manage and analyze banking complaints using MySQL.

The system stores customer complaints, assigns them to employees, tracks complaint status changes, and provides analytics for banking operations.

## Database Features

* 10+ normalized relational tables
* Foreign key relationships
* Stored procedures
* Triggers for complaint history tracking
* Complaint assignment system
* Analytical SQL queries
* Window functions

## Database Structure

Main tables include:

* customers
* banks
* complaints
* complaint_categories
* complaint_priority
* complaint_status
* employees
* departments
* complaint_assignments
* complaint_history

## Analytics Queries

The system provides analytics such as:

* Top banks with most complaints
* Monthly complaint trends
* Complaint category distribution
* Employee workload analysis
